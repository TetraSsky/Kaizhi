Création de la BDD :
CREATE DATABASE kaizhi_db
USE kaizhi_db;

Pour créer la table une fois la base de donnée créée : 
CREATE TABLE games
(
    gameid int NOT NULL, 
    titre text NOT NULL, 
    description text, 
    prix int, 
    image mediumblob, 
    lien_steam varchar, 
    PRIMARY KEY (gameid)
)

Vu que "gameid" est une clé primaire, on met en auto increment (Pour éviter réalistiquement des problèmes dans le futur avec des doublons) :
ALTER TABLE games MODIFY gameid INT AUTO_INCREMENT
